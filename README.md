# MyFirstProject
Coffee Shop prices and orders
# Lab: LabCoffeOrder 

This program displays drinks and prices, asks for your order, 
and displays your bill.

## How to Run
1. Open the program in your IDE.
2. Enter your drink order.
3. View the bill.
